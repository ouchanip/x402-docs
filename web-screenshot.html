Web Screenshot API

What it does

Captures screenshots of any webpage. Returns high-quality PNG images with configurable viewport and options.

Endpoint

POST https://web-screenshot-api-x402-689670267582.us-central1.run.app/capture

Price

$0.03 USDC (Base L2)

Request Example

{
  "url": "https://example.com",
  "width": 1280,
  "height": 800,
  "fullPage": false
}

Parameters

  • url - Target webpage URL (required)
  • width - Viewport width in pixels (default: 1280)
  • height - Viewport height in pixels (default: 800)
  • fullPage - Capture full scrollable page (default: false)

Response

Returns the screenshot as binary PNG data (image/png).
