Image OCR API

画像からテキストを抽出（OCR・ドキュメント解析）

基本情報

  • URL: https://image-ocr-api-x402-689670267582.us-central1.run.app
  • ネットワーク: Base Sepolia (eip155:84532)
  • Facilitator: https://x402.org/facilitator

エンドポイント

POST /ocr

画像からテキストを抽出します。

$0.05 USDC

リクエスト例
{
  "image": "data:image/png;base64,iVBORw0KGgo...",
  "language": "ja"
}
レスポンス例
{
  "text": "領収書\n株式会社サンプル\n金額: ¥10,000",
  "confidence": 0.95,
  "blocks": [
    { "text": "領収書", "bounds": { "x": 100, "y": 50, "width": 200, "height": 40 } }
  ]
}

POST /document

$0.10 USDC

リクエスト例

  "type": "receipt"
}

      { "name": "お茶 500ml", "quantity": 1, "price": 150 }
    ],

  },
